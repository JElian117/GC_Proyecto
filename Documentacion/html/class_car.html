Detailed Description

Esta clase es usada para crear y dibujar un objeto Car, para manejar el view (POV) de la escena, manejar la proyección y proporcionar los métodos necesarios para simular el movimiento del carro que son setters para la velocidad y dirección en cuestión de manejo del volante, un método para actualizar la posición del carro y un getter de la velocidad del carro.
